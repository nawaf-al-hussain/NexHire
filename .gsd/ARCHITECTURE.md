## Data Flow

1. **User Action:** User interacts with React Dashboard (e.g., clicks "Hire").
2. **API Request:** Frontend sends `axios.post` to `/api/recruiter/hire`.
3. **Controller Execution:** Express route handles request, validates session/role.
4. **SQL Execution:** Server calls `EXEC sp_HireCandidate` via `db.js`.
5. **Database Logic:** SQL Procedure validates vacancies, updates state, and logs changes via triggers.
6. **Response:** Server returns result back to Frontend for UI update.
